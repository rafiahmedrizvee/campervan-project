# 🚐 CamperStore – Campervan Accessories E-Commerce Website

A modern, fully responsive campervan accessories e-commerce website built with **React.js**, **Tailwind CSS**, and **GSAP animations**.

Designed for performance, smooth user experience, and premium UI interactions.

## 📌 Project Overview

CamperStore is an EU-focused online store for campervan accessories and van life essentials.  
The platform allows users to explore product categories, browse brands, read blogs, and shop essential gear for layout, interior, exterior, and vehicle technology.

## 🚀 Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/camperstore.git
```

### 2️⃣ Navigate to Project

```bash
cd camperstore
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Install Required Animation Library

```bash
npm install gsap
```

### 5️⃣ Start Development Server

```bash
npm run dev
```
